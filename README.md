# 🕵 Доска расследования

Интерактивная доска связей (детективный стиль). Один статичный файл `index.html` — без сборки и зависимостей. Открывается в любом браузере и деплоится на Cloudflare Pages как есть.

---

## 📁 Что в репозитории

```
.
├── index.html      ← вся доска (HTML + CSS + JS в одном файле)
└── README.md       ← этот файл
```

Все данные (карточки, связи, источники, таймлайн) лежат прямо внутри `index.html` в JS-объектах в самом верху тега `<script>`:

- `ZONES` — рамки-группы (Academix Hub, Larp Factory и т.д.)
- `N` — узлы/карточки (люди и проекты) со всеми полями и фактами
- `E` — связи (рёбра) между узлами
- `TL` — таймлайн внизу

Чтобы поменять контент — редактируется только эти четыре блока. Дизайн менять не обязательно.

## 🚀 Как залить в GitHub (один раз)

1. Создай пустой репозиторий на GitHub (например `detective-board`), **без** README/gitignore.
2. В папке с файлами выполни:

```bash
git init
git add .
git commit -m "init: доска расследования"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЛОГИН/detective-board.git
git push -u origin main
```

3. Дай другу доступ: **Settings → Collaborators → Add people** (введи его GitHub-ник). Тогда он сможет пушить в `main` напрямую.

## ✏️ Как вы оба правите через Claude и пушите

Самый удобный путь — **Claude Code** (правит файлы и сам делает git-команды).

**Первый раз (на каждой машине):**
```bash
git clone https://github.com/ТВОЙ_ЛОГИН/detective-board.git
cd detective-board
```

**Каждый раз перед работой** — подтянуть свежее, чтобы не было конфликтов:
```bash
git pull
```

**Дальше просто проси Claude в этой папке**, например:
> «Добавь на доску новый узел: человек X, ник @x, связан с Игорь Blink, достоверность Средняя, источник такой-то»

Claude отредактирует `index.html`. После этого:
```bash
git add .
git commit -m "добавил узел X"
git push
```

Cloudflare подхватит пуш и передеплоит за ~30–60 сек.

> ⚠️ **Важно про двоих:** всегда делай `git pull` **перед** правками. Если вы оба меняете один и тот же `index.html` одновременно, Git может выдать конфликт. Тогда либо договоритесь не редактировать внахлёст, либо разрешайте конфликт (Claude тоже умеет это делать — покажи ему сообщение об ошибке).

## ☁️ Деплой на Cloudflare Pages (один раз настроить → дальше авто)

1. Зайди на **dash.cloudflare.com** → слева **Workers & Pages** → **Create** → вкладка **Pages** → **Connect to Git**.
2. Авторизуй GitHub, выбери репозиторий `detective-board`.
3. Настройки сборки — **оставь пустыми**, это критично для статики:
   - **Framework preset:** `None`
   - **Build command:** *(пусто)*
   - **Build output directory:** `/`  (корень репозитория)
   - Production branch: `main`
4. Жми **Save and Deploy**.

Через минуту получишь адрес вида `https://detective-board.pages.dev`.

**С этого момента деплой автоматический:** любой `git push` в `main` (хоть твой, хоть друга, хоть через Claude) → Cloudflare сам пересобирает и обновляет сайт. Ничего больше нажимать не надо.

### Свой домен (опционально)
В проекте Pages → **Custom domains** → добавь свой домен и пропиши CNAME у регистратора (Cloudflare подскажет точную запись).

## 🎨 Подсказки по правке контента

Открой `index.html`, найди блок `const N = {` — это карточки. Шаблон одного узла:

```js
имя_узла:{
  t:'p',                       // 'p' человек · 'proj' проект · 'media-c' медиа
  conf:'hi',                   // достоверность: 'hi' Высокая · 'md' Средняя · 'lo' Низкая
  name:'Имя Фамилия',
  hd:'@ник',
  ph:'И',                      // буква на «фото»
  rot:-2,                      // наклон карточки в градусах
  x:680, y:80,                 // позиция на доске
  role:'Кто это в одну строку',
  stamp:['ok','ШТАМП'],        // ok / q / dead + текст печати (можно убрать строку)
  facts:[
    // [тип, ярлык, текст, 'Название источника', 'ссылка']
    // тип: 'ok' подтверждено · 'self' со слов · 'dead' опровергнуто · 'media' реклама/PR
    ['ok','Подтверждено','Текст факта.','Astana Hub','astanahub.com/...'],
  ],
  note:'Курсивная заметка сверху досье (необязательно)'
},
```

Связь добавляется в массив `const E = [`:
```js
// [откуда, куда, 'подпись на нити', стиль, достоверность]
// стиль: 'green' факт · '' обычная связь · 'media' пунктир-медиа · 'dashed' со слов
// достоверность: 'hi' · 'md' · 'lo'
['igor','eurasian','инвестиция $30k в TabAI','green','hi'],
```

> Главное правило проекта: **никаких фактов без источника**. Если подтвердить нечем — ставь `conf:'lo'` и тип `'dead'` с ярлыком «Не подтверждено».
